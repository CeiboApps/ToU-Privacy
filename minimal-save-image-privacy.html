Minimal Save Image

Privacy Policy

Last updated: 14 July 2026

Images never leave your browser. The only network call this extension ever makes is a license key check to api.polar.sh — and only if you buy Pro. No personal data, no usage data, no analytics, no crash reports, no advertising identifiers.

What the extension does with images

When you right-click an image and choose a format, the extension re-downloads that image from the page you are already on, decodes it with Chrome's built-in canvas, re-encodes it in the format you picked, and hands the result to Chrome's download manager. All of that happens inside your browser. The image is never uploaded anywhere.

What is stored, and where

DataWhereWhy
Your settings — format, quality, subfolder, "ask where to save" chrome.storage.local, on your machine So the extension remembers your preferences.
A count of images you've converted chrome.storage.local, on your machine Shown to you in the popup. Never transmitted.
Pro license key + activation ID chrome.storage.local, on your machine Cached so Pro features work offline. Only the key and an opaque activation ID are stored — no name, email, or card details.

Uninstalling the extension deletes all of it. None of it ever leaves your computer, and the developer cannot see any of it.

Payments

Pro features are unlocked via a one-time payment processed by Polar, which acts as the Merchant of Record. Card processing is handled by Stripe. This extension never sees your card details.

What happens during a Pro purchase:

  • You purchase a license on Polar's checkout page. A license key is sent to your email.
  • You paste the key into the extension's Pro page to activate it.
  • The extension calls api.polar.sh to activate and validate the key. The only data sent is the key itself and an opaque activation ID.
  • The paid status is cached locally so Pro features work offline. The key is re-validated once per day.

Each license key allows up to 2 device activations. You can deactivate a device to free up a slot.

Image conversion remains entirely local, on-device, with zero network calls. The license check is the only network call this extension makes, and only if you purchase Pro.

Permissions, in plain terms

PermissionWhat it's for

Third parties

The only third party is Polar (polar.sh), contacted solely for Pro license key activation and validation. No analytics providers, no ad networks, no affiliate redirects, no remotely-hosted code. Nothing is sold or shared.
